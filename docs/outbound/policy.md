# Outbound Policy

Last updated: 2026-05-21

## First outbound format

Initial outbound should use role-specific sample notes by league cluster, not fully custom club reports.

Current sample note categories:

- Japan U25 Central Midfielder
- Japan U25 Wide Attacker / Attacking Midfielder
- Japan U25 Side Centre-Back / Left Centre-Back

## Current structure

- cover
- why this role / Japan market context
- sample profile 01
- profile to monitor
- what Japan Talent Desk can verify next / contact

Safer first-outbound structure:

- 1 primary profile
- 1 monitor profile
- clear verification flags
- no final verdict

## CRM policy

Use:

- `PrimaryAddress`
- `SecondaryAddress`
- `GeneralAddress`

Guessed emails can be used for low-volume testing but must be marked:

- `Guessed`
- `needs_verification`

## Send pattern

- first batch: 20 to 30 clubs
- do not blast the full list at once
- track bounces, replies, wrong-person referrals, and note-specific interest

## Primary CTA

If this type of Japan-side screen is useful, Japan Talent Desk can prepare a short club-specific note around one current role need.
