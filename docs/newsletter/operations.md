# Japan Market Weekly Operations

Last updated: 2026-05-21

## Positioning

Japan Market Weekly is a short weekly recruitment note for European clubs monitoring the Japanese market.

This is a secondary trust-building asset, not the main CTA in first outbound.

## Current tooling

- platform: Brevo
- purpose: list management, forms, tags, welcome automation, and scheduled sends

## Editorial rule

Do not write generic match summaries.

Every item should answer:

- what changed
- why it matters for recruitment
- what should be verified

## Automation boundary

Allowed:

- AI-assisted draft
- Sei review
- scheduled delivery

Not allowed:

- fully automated scrape
- AI write
- send without verification

## Default sections

- three Japan-side signals this week
- two U25 profiles to monitor
- one role-specific note
- injury / availability watch
- transfer / loan / contract signal
- youth / national-team note
- Japan-side context note
