# Report Principles

Last updated: 2026-05-21

## Proof case

RAAL La Louviere is the current proof case for club-facing report style.

## Core report rules

- narrow
- brief-specific
- practical
- not a generic Japanese player list
- clear risk flags
- no fee or salary estimates
- no unsupported speed claims
- contract and availability treated as verification items
- final slide includes contact and next-step value

## Usage rule

The RAAL-style output is the structural source-of-truth for club-facing reports.

Outbound sample notes are lighter than full club reports, but they should inherit the same discipline around realism, risk flags, and verification items.
