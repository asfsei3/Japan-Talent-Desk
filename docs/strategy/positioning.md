# Japan Talent Desk Positioning

Last updated: 2026-05-21
Status: Current working source-of-truth

## Core position

Japan Talent Desk provides Japanese market recruitment intelligence for European football clubs.

The value is not only naming Japanese players.

The value is translating Japan-side context into practical recruitment decision support for European clubs.

## Do not position JTD as

- agent
- broker
- generic scouting database
- AI player recommendation platform
- video scouting service
- longlist provider

## Position JTD as

- Japan-side context layer
- local-source translator
- recruitment risk filter
- availability and deal-realism interpreter
- player-side signal reader
- practical next-step desk for European clubs

## External wording priorities

Prefer:

- initial role-specific screen
- sample profile
- profile to monitor
- what to verify
- contract status requires direct confirmation
- recent first-team rhythm
- player-side openness to Europe
- availability should be verified directly

Avoid:

- final recommendation
- hidden gem
- perfect fit
- guaranteed availability
- risk-free
- cheap
- unsupported comparison language

## Core note

Use this framing where useful:

This is an initial role-specific screen, not a final recruitment recommendation. Transfer fee, salary, availability, and physical benchmarks should be treated as verification items.
